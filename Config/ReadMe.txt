--------- Config ---------
Définition des différentes classes, qui vont remplacer toutes les fois ou les différentes "class" sont appellées sauf la création.

Définition des tables dans MySql, qui vont remplir les différentes tables dans toutes les requetes.

Deux variables différentes au cas ou la table porte un nom différent de la classe ou si la table
ne comporte pas de majuscule.
